# Ewing Initialization App

A module rendering environment initialization frameworks independently
out of standard runtime constraints. Evaluates base configurations via
\`init.simulation()\`.

## Usage

``` r
initApp(title = "Population Ethology")

initServer(id, init_par)

initOutput(id)
```

## Arguments

- title:

  Application title

- id:

  module ID string

- init_par:

  list of natively structured initialization counts (e.g. \`\$host\`,
  \`\$parasite\`)
