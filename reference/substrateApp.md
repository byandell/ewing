# Substrate Plot App

A focused graphical module aggregating structural species progression
separated distinctly across physical environment thresholds mapped by
evaluating top-level topological definitions.

## Usage

``` r
substrateApp()

substrateServer(id, simres)

substrateOutput(id)
```

## Arguments

- id:

  module ID string

- simres:

  reactive object resolving chronological population metrics

- title:

  Application title
