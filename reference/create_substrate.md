# Create Substrate Geometric Object

Iterates against a defined configuration metric to build explicit
spatial network definitions tracking Euclidean point plots, dynamic
border labels, and spanning geometric boundary polygons.

## Usage

``` r
create_substrate(topology, width = 10, step = 1)
```

## Arguments

- topology:

  Abstract parameter list derived from \`substrate_topology()\`.

- width:

  Physical geometric limits of bounding constraints.

- step:

  Inner plotting density logic parameter.

## Value

An S3 object of format class \`substrate\`, storing resolved attributes.
