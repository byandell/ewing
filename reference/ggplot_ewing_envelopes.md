# GGplot of Ewing multiple envelopes

GGplot of Ewing multiple envelopes

## Usage

``` r
ggplot_ewing_envelopes(object, confidence = FALSE, main = "", ...)
```

## Arguments

- object:

  object of class \`ewing_envelope\`

- confidence:

  plot confidence bands if \`TRUE\`

- main:

  title for plot

- ...:

  additional arguments

- cols:

  colors for top simulations
