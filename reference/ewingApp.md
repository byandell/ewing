# Ewing App

Core \`Systems Ethology\` Shiny platform acting as the central
interactive wrapper modularizing age distributions, probability
envelopes, and runtime data interactions.

## Usage

``` r
ewingApp(title = "Systems Ethology")

ewingServer(id)

ewingInput(id)

ewingOutput(id)
```

## Arguments

- title:

  Application title

- id:

  module ID string
